Documentation site for [Smart Wallet](https://github.com/coinbase/smart-wallet) and using it with the [Coinbase Wallet SDK](https://github.com/coinbase/coinbase-wallet-sdk).

### Running locally

```bash
bun install 
bun run dev
```
